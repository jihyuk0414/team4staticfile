HTML5이란?

HTML5는 Hyper Text Markup Language 5의 약자로, 웹 페이지를 만들고 구성하는 데 사용되는 최신 버전입니다. HTML5는 웹 기술의 발전과 함께, 더 풍부하고, 기능적이며, 모바일 친화적인 웹 페이지와 애플리케이션을 만들 수 있도록 다양한 기능을 추가한 버전입니다.

HTML5의 주요 기능

  • 새로운 구조적 태그: <header>, <footer>, <article> 등
  • 멀티미디어 지원: <audio>, <video>
  • 폼 입력 기능 향상: <input type="email">, <input type="date">, <input type="tel"> 등
  • 그래픽 기능: <canvas>, <svg>
  • API 지원:
    • Web Storage: 로컬 저장소를 사용해 클라이언트 측 데이터를 저장할 수 있음.
    • Geolocation: 사용자의 위치 정보를 얻을 수 있음.
    • Web Workers: 백그라운드 스레드를 사용해 웹 애플리케이션의 성능을 향상시킬 수 있음.
  • 모바일 최적화: 뷰포트 설정을 통해 모바일 장치에서 웹 페이지를 최적화하여 보여줄 수 있음.
  • 간단한 문서 선언: <!DOCTYPE html> 으로 문서 선언이 단순화되어, 코드가 깔끔하고 이해하기 쉬워짐.

HTML5의 기본 구조

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 실제 웹 페이지 내용 -->
</body>
</html>
  • <!DOCTYPE html>: 문서가 HTML5 문서임을 선언함.
  • <html>: HTML 문서의 시작과 끝을 나타냄.
  • <head>: 문서의 메타정보(문서 제목, 문자 인코딩 등)를 포함함.
  • <body>: 실제 웹 페이지의 내용이 포함되는 부분.
