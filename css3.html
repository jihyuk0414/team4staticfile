CSS 스터디 페이지

CSS란?

(해당페이지는 이미지 없이, CSS만으로 구현되었습니다)

CSS(Cascading Style Sheets)는 웹 페이지의 시각적인 표현을 담당하는 스타일 시트 언어입니다. HTML이 웹 페이지의 구조와 내용을 정의한다면, CSS는 그 내용의 디자인과 레이아웃을 결정합니다.

CSS
→

CSS의 주요 기능:

  • 글자 스타일링 (글꼴, 크기, 색상, 간격 등)
  • 레이아웃 제어 (위치, 크기, 여백 등)
  • 시각적 효과 (그림자, 투명도, 배경 등)
  • 반응형 디자인 구현 (다양한 화면 크기에 적응)
  • 애니메이션 효과

CSS는 "선택자(selector)"를 사용해 HTML 요소를 선택하고, "속성(property)"과 "값(value)"으로 이루어진 선언(declaration)을 통해 스타일을 적용합니다.

selector {
  property: value;
}

CSS3와 기존 CSS의 차이

CSS3는 CSS의 세 번째 주요 버전으로, 이전 버전들에 비해 크게 향상된 기능을 제공합니다.

주요 차이점:

  1. 모듈식 구조:

    CSS3는 여러 독립적인 모듈로 구성되어 있어 브라우저가 점진적으로 기능을 지원할 수 있게 되었습니다.

  2. 새로운 시각적 효과:

    • 그림자 효과 (box-shadow, text-shadow)
    • 둥근 모서리 (border-radius)
    • 그라데이션 (linear-gradient, radial-gradient)
    • 투명도 (opacity, rgba())

  3. 향상된 레이아웃 기능:

    • 플렉스박스 (간편한 배치, 정렬)
    • 그리드 레이아웃 (2차원 배치, 정렬)
    • 다중 열 레이아웃 (여러 열 자동 배치, 정렬)

  4. 애니메이션과 변환:

    • 애니메이션 (@keyframes, animation)
    • 변환 효과 (transform, transition)
    • 3D 효과

  5. 미디어 쿼리:

    • 화면 크기, 해상도에 따른 반응형 (자동 맞춤형) 디자인 구현

기존 CSS가 기본적인 스타일링을 위한 것이라면, CSS3는 복잡한 레이아웃과 애니메이션 효과를 간편하게 구현할 수 있게 해주었습니다.

CSS 박스 모델

CSS 박스 모델은 요소의 크기와 공간을 제한하는 개념입니다. 모든 HTML 요소는 사실 박스안에 있으며, 다음 네 가지 영역으로 구성됩니다:

  1. 콘텐츠 영역(Content): 텍스트, 이미지 등 실제 내용의 영역
  2. 패딩(Padding): 콘텐츠와 테두리 사이의 공간
  3. 테두리(Border): 패딩을 둘러싸는 영역
  4. 마진(Margin): 테두리 바깥쪽의 공간, 다른 요소와의 거리

              }

정렬

CSS에서 요소를 정렬하는 방법은 여러 가지가 있습니다.

왼쪽 정렬 글자

.text { text-align: left; }

가운데 정렬 글자

.text { text-align: center; }

오른쪽 정렬 글자

.text { text-align: right; }

텍스트 정렬 예시 Code

.text {
  text-align: left | right | center | justify;
}

인라인 요소 수직 정렬 예시 Code

.inline {
  vertical-align: baseline | top | middle | bottom;
}

블록 요소 수평 정렬 예시 Code

.block {
  margin: 0 auto; 
}

Flexbox를 이용한 정렬 예시 Code

CSS3에서 추가된 기능인 Flexbox는 1차원 레이아웃 시스템으로, 보기 쉬운 코드 형태로 전체를 정렬하게 도와줍니다.

.container {
  display: flex;
  
  /* 가로를 축으로 잡고 정렬 */
  justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
  
  /* 세소를 축으로 잡고 정렬 */
  align-items: flex-start | flex-end | center | baseline | stretch;
  
  /* 여러 방향 기준 정렬 */
  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}

전환(Hover) 예시 Code

.hover-button {
                padding: 12px 24px;
                background-color: #4a6da7;
                color: white;
                border: none;
                border-radius: 5px;
                cursor: pointer;
                transition: transform 0.3s, background-color 0.3s, box-shadow 0.3s;
              }

              }
